# Portfolio React

Single-page portfolio website built with React, TypeScript, and Vite.

## Sections

- Hero
- About
- Projects
- Skills
- Contact

## Customize Content

Replace placeholders in these files:

- `src/App.tsx` for text and links
- `src/App.css` and `src/index.css` for styling

The social links, email address, and project details are placeholders and should be replaced with your real information.

## Development

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the build:

```bash
npm run preview
```
